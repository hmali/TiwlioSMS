# GMADP Logo

Place the GMADP logo file in this directory as:
- `gmadp-logo.png` - Main logo file

The logo should be:
- Format: PNG with transparent background (recommended)
- Size: Recommended 200x200px or similar square ratio
- The logo will be displayed in the navbar at approximately 40px height

## Current Logo
The attached logo shows "Shivam Maharaj America Defense Fund" emblem - this will be used as the GMADP branding logo.

## Usage
The logo is referenced in:
- `templates/base.html` - Main navigation bar
- `templates/login.html` - Login page header
